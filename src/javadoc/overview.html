Description:

Project contains API Wrapper for old Deprecated Kraken API which contains only V5, and new API called Helix

Twitch4J it is not only API wrapper. It is a WebHook, OAuth2 (with Open ID Connect), IRC and PubSub in one library.

This project is still in development, check out Features and Changelog to follow progress.
